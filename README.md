### 功能

- 支持 `org.springframework.data.domain.Page` 的序列化和反序列化
- 适用于 `OpenFeign`

### 使用方法

- 在 `pom.xml` 中添加依赖：

```xml
<dependency>
    <groupId>io.github.benfromchina</groupId>
    <artifactId>jackson-datatype-page</artifactId>
    <version>1.0.0</version>
</dependency>
```
